## `> about`

**PENNY** is a personal finance coach that analyzes your expenses to give you a an overview of where that money went and what you can do in the future to save it. Built with Python, Groq (LLaMA 3.1), and Streamlit..

No spreadsheets. No sign-ups. Just paste your expenses and get an instant breakdown with saving tips.

## `> install`

```bash
# clone
git clone https://github.com/rahimah-gilani/finance-gal-penny.git
cd finance-gal-penny

# create and activate virtual environment
python -m venv venv
source venv/bin/activate        # Windows: venv\Scripts\activate

# install dependencies
pip install -r requirements.txt

# add your Groq API key
cp .env.example .env
# then open .env and paste your key
```
